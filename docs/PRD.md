# PRD

## 요구사항 문서

- Product Requirements Document

### 1순위(필수)

#### 사용자
- 회원가입_이메일, 아이디, 비밀번호, 닉네임을 입력하여 회원가입
- 로그인_소셜로그인, 아이디/비밀번호를 이용한 로그인
- 로그아웃_현재 로그인 세션 종료

- 상품 등록_제목, 가격, 카테고리, 거래지역, 상품상태, 설명, 이미지 등록
- 상품 목록 조회_등록된 중고 상품 목록 조회
- 상품 상세 조회_상품 이미지, 설명, 가격, 판매자 정보, 거래 상태 확인
- 상품 수정_본인이 등록한 상품 정보 수정
- 상품 삭제_본인이 등록한 상품 삭제
- 상품 이미지 업로드_상품 사진 여러 장 등록
- 카테고리별 조회_전자기기, 의류, 도서, 생활용품 등 카테고리별 상품 조회
- 상품 검색_키워드로 상품 검색
- 가격 필터_최소 가격, 최대 가격 기준으로 필터링
- 지역 필터_거래 가능 지역 기준으로 필터링
- 최신순 정렬_최근 등록된 상품 순으로 정렬
- 가격순 정렬_낮은 가격순, 높은 가격순 정렬
- 거래 상태 변경_판매중, 예약중, 판매완료 상태 변경

- 게시글 작성_사용자가 자유롭게 커뮤니티 글 작성
- 게시글 목록 조회_커뮤니티 게시글 목록 조회
- 게시글 상세 조회_게시글 제목, 내용, 작성자, 작성일, 조회수 확인
- 게시글 수정_작성자 본인만 게시글 수정
- 게시글 삭제_작성자 본인만 게시글 삭제
- 게시글 검색_제목 또는 내용 기준 검색
- 게시글 카테고리_자유게시판, 동네정보, 거래후기, 질문게시판 등 분류

- 댓글 작성_게시글에 댓글 작성
- 댓글 목록 조회_게시글별 댓글 목록 조회
- 댓글 수정_작성자 본인만 댓글 수정
- 댓글 삭제_작성자 본인만 댓글 삭제
- 댓글 수 표시_게시글 목록에 댓글 개수 표시

- 게시글 좋아요_마음에 드는 게시글에 좋아요
- 좋아요 취소_이미 누른 좋아요 취소

- #### 관리자
- 회원 목록 조회_가입된 회원 목록 확인
- 회원 상세 조회_회원 정보, 작성 글, 상품, 신고 내역 확인
- 회원 상태 변경_정상, 정지, 탈퇴 처리
- 관리자 권한 관리_일반 사용자와 관리자 구분

- 전체 상품 조회_모든 상품 목록 조회
- 상품 상세 조회_등록된 상품 상세 정보 확인
- 부적절한 상품 삭제_신고되거나 정책 위반 상품 삭제
- 거래 상태 확인_판매중, 예약중, 판매완료 상태 확인
- 카테고리 관리_상품 카테고리 추가, 수정, 삭제

- 게시글 관리_전체 게시글 조회, 삭제
- 댓글 관리_전체 댓글 조회, 삭제
- 게시판 카테고리 관리_게시판 카테고리 추가, 수정, 삭제

### 2순위(선택)

#### 사용자
- 회원정보 조회_내 프로필, 닉네임, 이메일, 가입일 확인
- 회원정보 수정_닉네임, 프로필 이미지, 비밀번호 수정
- 회원 탈퇴_사용자가 직접 계정을 탈퇴
- 프로필 이미지 등록_사용자 프로필 사진 업로드
- 판매자 프로필 조회_다른 사용자의 판매 상품, 매너 점수, 거래 후기 조회

- 조회수 증가_상품 상세 페이지 조회 시 조회수 증가
- 내 판매글 조회_내가 등록한 상품 목록 조회

- 상품 찜하기_관심 있는 상품을 찜 목록에 추가
- 찜 취소_찜한 상품을 취소
- 찜 목록 조회_내가 찜한 상품 목록 확인
- 찜 개수 표시_상품 상세 페이지에 찜 개수 표시

- 신고 기능

- 거래 및 채팅 기능

- 매너 점수 반영_후기 또는 평가를 기반으로 사용자 신뢰도 반영

- 조회수 증가_게시글 상세 조회 시 조회수 증가
- 인기글 조회_조회수 또는 좋아요 수가 높은 게시글 표시

- 대댓글 작성_댓글에 대한 답글 작성

- 좋아요 수 표시_게시글별 좋아요 수 표시
- 인기 게시글 반영_좋아요 수를 기준으로 인기글 선정

- 알림기능
- 마이페이지

#### 관리자
- 인기글 관리_인기글 노출 여부 관리
